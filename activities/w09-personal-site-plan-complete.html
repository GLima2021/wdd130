Your personal Site Plan - Completed

Activity Instructions

Estimated Time: 45 minutes

This week along with finishing up the rafting site, we need to finish up the planning for our personal site.

Information about your personal site that might be helpful to know as you finish the personal site plan.

By the end of this course:

  1. The personal site must consist of a home page and at least 2 child pages.
  2. The Web site must be hosted on the web.
  3. Each page of your Web site must satisfy the following requirements:
    1. Significant textual content consisting of appropriate content for your audience so that they can understand the meaning and message of this site.
    2. At least one image that relates to the content of the page.
    3. Working links to your two child pages.
    4. Be laid out and styled according to your site plan wireframes.
    5. Be visually appealing and easily readable.

This week we only need to finish the planning for the personal site. We will write the HTML and CSS to build the site over the rest of the semester.

Activity Directions

  1. Create the Home wireframe

    If you have completed the previous personal site plan assignments, you should only have the 3 wireframes to complete for each of your pages. Review the content you prepared last week for each page that you are planning to build, then start with the Home page. Make sure you account for each block of content in your plan. Pay special attention to the header and footer, as these should be on each page. Keep in mind as well the principles of good design.

  2. Create the other wireframes

    Create two more wireframes for the subpages. Use the same header and footer design as the home page for each of these pages.

  3. Verify

    An image of each wireframe should be placed in the wireframe section of your personal site plan. Review each section of your site plan. Make sure that each is complete and free of spelling and grammar errors. Then submit the URL of your personal site plan to Ilearn.
