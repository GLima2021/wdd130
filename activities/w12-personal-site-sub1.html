WDD 130

Personal Site Subpage 1

Last week, you should have finished styling your Home page. This week we will style the first child page.

Activity Directions

  1. Open the file

    Open the file for your first subpage.

  2. Child Page Grid

    Open the wireframe for your first child page. We need to build the Grid for your layout. From the wireframe, determine how many columns and rows you will need. Create those rows and columns with CSS, then start placing your elements into the grid cells.

  3. Spacing and Alignment

    Add padding and margins where appropriate to give your layout the right amount of white-space.

    Adjust the alignment of elements as well at this time as needed.

  4. Link it up

    Make sure to update the navigation in both your Home page and the child page so that if you select the link on either page, it will take you to the other.

    Note that this week, only the first child page is expected to be complete. You will need to follow the steps above for the second child page as well, but it is due with the completed site next week.

  5. Publish

    Validate your HTML and CSS, fix any errors, then submit the URL to your site in I-Learn. Make sure the URL takes the instructor to your child page that is completed.
