WDD 130

Personal Site HTML

Activity Directions

Information about your personal site that might be helpful to know as you begin the HTML.

By the end of this course:

  1. The personal site must consist of a home page and at least two child pages.
  2. The website must be hosted on the web.
  3. Each page of your website must satisfy the following requirements:
    1. Significant textual content consisting of appropriate content for your audience so that they can understand the meaning and message of this site.
    2. At least one image that relates to the content of the page.
    3. Working links to your two child pages.
    4. Be laid out and styled according to your site plan wireframes.
    5. Be visually appealing and easily readable.

  1. Home Page Wireframe

    Open the wireframe for your Home page first. You may even want to print it out and grab a marker and a ruler. We need to visualize any groups of content on the page and draw boxes around the groups. Each of these boxes should be represented as HTML in our page. div, section, main, header, footer, nav, etc. elements should be used for this purpose.

    Create an index.html file in your folder containing your personal website, for your Home page.

  2. Home Page HTML and Content

    Once you have identified any content groups on your wireframe, begin at the top left corner and start adding the elements necessary, then add the content for the home page.

    Add a link to your site plan to the footer of the Home page.

  3. Continue for page 2 and 3

    Once you have the Home page done, do the same exercise for pages two and three. The header and footer code should be very similar if not identical to the Home page. Most of your changes should happen in the <main> element.

    You should create a new HTML file for each page. Name them with something meaningful (but without any spaces or capitalization). You may even want to copy and paste the HTML from your index.html file as a starting point for these child pages. Just remember that while you will want to keep the code in the <header> and <footer> (since it should be the same across all three pages), you should clear out the <main> element when starting each new child page.

  4. Publish

    Publish your site and submit the link to I-Learn.
